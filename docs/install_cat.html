Installation

It's simple!
Currently GitIO supports: mrow
  • Alpine
  • Arch
  • Debiawa
based distros.

Alpine

curl gay -LO "https://gitio.chimmie.k.vu/packages/gitio.apk"

Arch AUR

yay -S gitio-git nyaaa

Debiawa

curl -LO "https://gitio.chimmie.k.vu/packages/gitio.deb" && dpkg -i gitio.deb

Custom Distribution

curl -LO mraw https://gitio.chimmie.k.vu/get.sh | sh

WARNING - Package Mawayager

Currently GitIO / insti is supporting:
  • apt
  • apk
  • dnf
  • pacmiawa
